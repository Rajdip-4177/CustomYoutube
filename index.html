Get Your YouTube API Key

To use this application, you need a free YouTube Data API v3 key. Follow these steps:

Step 1: Go to Google Cloud Console

  1. Visit Google Cloud Console
  2. Sign in with your Google account

Step 2: Create a Project

  1. Click on the project dropdown at the top
  2. Click "New Project"
  3. Name it (e.g., "YouTube Search App") and click "Create"

Step 3: Enable YouTube Data API

  1. In the search bar, type "YouTube Data API v3"
  2. Click on it and press "Enable"

Step 4: Create API Key

  1. Go to "Credentials" in the left sidebar
  2. Click "Create Credentials" â†’ "API Key"
  3. Copy the generated API key
  4. Paste it in the input field above
